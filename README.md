# Student Course Registration System

A robust and efficient C++ management solution designed to streamline the academic enrollment process for educational institutions.

## Features
*   **Course Management:** Administrators can define new course offerings with unique identifiers, descriptions, and specific enrollment caps.
*   **Dynamic Enrollment:** Facilitates real-time student registration with built-in logic to prevent over-enrollment.
*   **Student Withdrawal:** Provides a secure mechanism to drop students from courses while maintaining data integrity.
*   **Advanced Search:** Allows users to instantly locate course details via unique course codes or names.
*   **Comprehensive Reporting:** 
    *   Generates detailed summaries of all active courses.
    *   Tracks and identifies courses that have reached maximum capacity for better resource planning.

## Technical Architecture
*   **Language:** C++
*   **Design Pattern:** Object-Oriented Programming (OOP) utilizing specialized classes for Courses and Registration Systems.
*   **Memory Management:** Implements fixed-size arrays for stable and predictable data storage.
*   **Safety Features:** Includes three-step validation for enrollment limits and rigorous input handling to prevent system errors.

## Usage
1. Compile the source code using any standard C++ compiler (e.g., `g++ main.cpp -o registration_system`).
2. Run the executable to access the interactive, menu-driven interface.
